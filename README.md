# Vehicle-Detection-System-Using-OpenCV



## Overview

This project is a real-time vehicle detection and tracking system developed using Python and OpenCV. The system detects moving vehicles from traffic video footage and tracks them using unique IDs and bounding boxes.


## Features

- Real-time vehicle detection
- Object tracking using unique IDs
- Green bounding box detection
- Background subtraction using MOG2
- Traffic monitoring using Computer Vision


## Technologies Used

- Python
- OpenCV
- NumPy

  
## Project Output

The system detects moving vehicles and highlights them using green bounding boxes with object tracking IDs.


## Files Included

- main.py
- tracker.py
- requirements.txt
- white-mask-output.png
- main-output.png
- demo.mp4 


## How to Run

1. Install required libraries:
pip install -r requirements.txt

2. Run the project:
python main.py
